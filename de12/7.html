🌦 天気予報アプリ（Python × OpenWeatherMap API）

1. OpenWeatherMap APIの説明

OpenWeatherMap は、世界中の気象データを無料で提供しているサービスです。
開発者向けにAPI（Application Programming Interface）を公開しており、Pythonなどのプログラムから 都市名を指定するだけでリアルタイムの天気情報を取得できます。

  • 世界中の都市の現在の天気に対応
  • 毎分60回までリクエスト可能（無料プラン）
  • JSON形式で気温・湿度・風速などの情報を取得可能

2. Pythonコードの例

以下のコードを VS Code に貼り付けて保存してください。ファイル名は weather_app.py にします。

# weather_app.py
# 天気予報アプリ（Python × OpenWeatherMap API）

import requests

# ===== 自分のOpenWeatherMap APIキーをここに入力 =====
API_KEY = "ここに自分のAPIキーを入れてください"
# ======================================================

BASE_URL = "http://api.openweathermap.org/data/2.5/weather"

def get_weather(city_name):
    """指定した都市の天気情報を取得して表示"""
    params = {
        "q": city_name,
        "appid": API_KEY,
        "lang": "ja",       # 日本語で出力
        "units": "metric"   # 摂氏（℃）で表示
    }

    response = requests.get(BASE_URL, params=params)

    # ステータスコード200なら成功
    if response.status_code == 200:
        data = response.json()
        weather = data["weather"][0]["description"]
        temp = data["main"]["temp"]

        print(f"\n🌤 {city_name}の現在の天気情報")
        print("------------------------------")
        print(f"天気：{weather}")
        print(f"気温：{temp}℃")
        print("------------------------------\n")

    else:
        print("❌ 都市名が見つかりませんでした。")


if __name__ == "__main__":
    print("=== 天気予報アプリ (OpenWeatherMap API) ===")
    city = input("都市名を入力してください: ")
    get_weather(city)
