第一回 ARDUINO

arduinoとは

Arduinoとは入出力ポートがあるマイコンボード。
プログラムを書いて、センサーの値を読み込んだり、モーターを動かしたり、LEDを光らせたりと電子工作といわれることが簡単にできる。

arduinoでLチカ

LチカとはLEDをチカチカさせること。

以下のようなプログラムでLチカ

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
