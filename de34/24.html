Flaskとは？

Flaskは、Pythonを使ってWebアプリケーションを開発できるフレームワークです。 軽量でシンプルな構造になっているため、 初心者でも扱いやすく、必要な機能を自由に追加できる特徴があります。

今回制作したプログラム

今回はCSVファイルを読み込み、 その内容をWebページに表示するプログラムを制作しました。 また、場所の情報をGoogleマップのリンクへ変換し、 クリックすると地図を開けるように工夫しました。

工夫した点

  • CSVデータをリスト形式で取得した
  • 位置情報をGoogleマップのURLへ変換した
  • 条件分岐を使い表示内容を変更した

データ表示

{% for data in mmlist %}
{% if data[1] == "場所" %} {{ data[1] }}
{% elif data|length > 3 and data[3] %} {{ data[1] }}
{% else %} {{ data[1] }}
{% endif %} {% endfor %}

まとめ

Flaskを使用することで、 Pythonによるデータ処理とWebページの表示を簡単に連携させることができました。 今回の制作を通して、普段利用しているWebアプリケーションが、 バックエンドとフロントエンドの2つの要素によって成り立っていることも理解できました。

また、条件によって表示内容を変化させる機能を作成したことで、 SNSやショッピングサイトなどの身近なWebサービスにも、 同じような仕組みが利用されているのではないかと感じました。

普段は利用者として見ることが多かったWebページの裏側を知ることができ、 PythonとHTMLを組み合わせて実際に動作するページを作れたことがとても面白かったです。 今後はデザイン面の工夫やデータベースとの連携にも挑戦し、 より実用的なWebアプリケーションを制作していきたいです。
